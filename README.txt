RevisaFacul - Landing Page

Arquivos:

- styles.css: estilos visuais e responsividade
- script.js: menu mobile
- assets/logo-revisafacul.png: logo usada no topo

Como usar:
1. Abra o index.html no navegador para visualizar.
2. Troque todos os textos LINK_DE_PAGAMENTO_AQUI pelo link real de pagamento.
3. Publique a pasta em Vercel, Netlify ou hospedagem estática.
